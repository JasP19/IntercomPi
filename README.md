# IntercomPi

## Requirements

Python is the main requirement for this project

## Setup

Create a `config.json` file with the following format:

```
{
  "unit1": {
    "numbers": [<primary phone number>, <secondary phone number>],
    "bill": 0
  },
  "unit2": {
    "numbers": [<primary phone number>, <secondary phone number>],
    "bill": 0
  }
}

```
